## Books Blog

## Description
This blog is a web application built using Django and HTML. It serves as a unique library where users can register, login, and access a collection of books. The application provides features such as browsing the book collection, viewing book details, and adding books to the cart. It does not include an API component.

## Installation
1. Clone the repository to your local machine.
2. Create a virtual environment using `virtualenv` or `conda`.
3. Activate the virtual environment.
5. Set up the database by running the necessary migrations with `python manage.py migrate`.
6. Start the development server using `python manage.py runserver`.
7. Access the application in your web browser at `http://localhost:8000`.

## Usage
- Register a new account by providing the required information.
- Login using your credentials to access the library.
- Browse through the collection of books and view details of each book.
- Add books to your cart for future reference.
- Logout when you are done using the application.
